=== (A) HOMOGRAPH/POLYSEMY ABSORPTION FRAMING: VERDICT = NOVEL (high confidence) ===
Documented feature absorption is empirically almost entirely a FIRST-LETTER SPELLING phenomenon. Chanin et al.'s 'A is for Absorption' (2409.14507, NeurIPS 2025 Oral) frames the failure mode via the word 'short' absorbing the 'starts with S' concept (Figure 1) and explains it as 'a consequence of hierarchical co-occurrence between sparse and dense features... optimizing for sparsity whenever the underlying features form a hierarchy' [2][3]. SAEBench (2503.09532, ICML 2025) operationalizes its Feature Absorption metric by building 'on the metric proposed in Chanin et al. (2024b), which examines how SAE latents represent FIRST-LETTER classification tasks', with the canonical 'starts with S fires on 95% of S-tokens, absorbed on the other ~5%' illustration [4]. The dominant MITIGATION line (Matryoshka SAEs, 2503.17547, ICML 2025) ties absorption to nesting/dictionary size [5]. So prior work predicts WHEN absorption occurs from STRUCTURAL levers (sparsity, hierarchy, dictionary size) measured on SPELLING -- never from token polysemy/homography on a SEMANTIC hierarchy.

The key near-miss, PS-Eval (Minegishi, Furuta, Iwasawa, Matsuo; arXiv:2501.06254; 'Published as a conference paper at ICLR 2025') [6], evaluates whether SAE features SEPARATE the senses of polysemous words (a WiC-style same-meaning/different-meaning Recall/Precision/Accuracy metric over whether the same sparse feature fires for the same sense), and finds deeper layers + attention help disambiguate. Crucially, a full-text grep over the PS-Eval PDF for 'absorption|recall hole|router|starts-with|first-letter|spelling|country|Georgia|Jordan|suppress' returns a SINGLE hit -- 'JumpReLU suppresses small activations' (the activation function), NOT a suppressed parent latent [7]. PS-Eval therefore never studies the absorption failure mode, never exhibits a suppressed parent or a recall hole, and provides no a-priori recall-hole router. CCRG's observation -- that absorption recurs on homograph/polysemous tokens whose general 'is-a-country' parent latent is suppressed (Georgia = US-state/country/given-name; Jordan = country/given-name), with the absorption-type slices being EXACTLY the homographs that show parent recall holes, flagged a priori by the router -- is a genuinely new empirical slice. PS-Eval is a clean cite-and-distinguish, not a precedent. FRAMING GUARDRAIL for the paper: state this as 'absorption recurs on polysemous tokens, predicted by the recall-hole signal,' NOT 'broad taxonomic generalization' (honest scope: two homographs on one taxonomic concept, consistent with the absorption mechanism). Short + long drop-in positioning paragraphs and the one-line PS-Eval distinguisher are in research_report.md section A.

=== (B) KG-LOCALIZED SURGICAL SUB-CONCEPT EDIT: VERDICT = DISTINCT (high confidence) ===
No prior SAE/LLM method performs CCRG's downstream edit: reading an INTERVENTIONAL feature-KG to find the single ABSORBER latent the graph NAMES as the specialist for ONE sub-context, editing THAT latent to surgically change/recover only that sub-context while PRESERVING the parent concept, measured as targeted recall-recovery vs a RANDOM-ADDITION control PLUS side-effect KL on unrelated prompts. SAE-TS (Chalnev, Siu, Conmy; 2411.02193) learns a linear effect-approximator and builds a steering vector s = M_j/||M_j|| - lambda*M_b/||M_b|| to push a TARGET SAE FEATURE up while minimizing aggregate side effects -- a concept feature chosen as 'the desired feature' with a tuned coefficient, not a KG-named absorber for a sub-context [10]. SRS (He et al.; 2503.16851) contrasts positive/negative prompt pairs and uses bidirectional KL divergence to locate dimensions for a target ATTRIBUTE (safety/fairness/truthfulness) on Gemma-2 -- attribute-level guardrails, not parent-preserving sub-context edits [11]. SALVE (Flovik; 2512.15938; ICLR 2026 Trustworthy-AI Workshop) performs weight-space edits of CLASS-DEFINING features validated on ResNet-18 and ViT-B/16 -- VISION models, a modality + unit-of-analysis distinguisher, not an LLM-SAE absorber [12][13]. LEACE (Belrose et al.; 2306.03819; NeurIPS 2023) is closed-form DENSE concept erasure that provably blocks all linear classifiers from a concept while changing the embedding minimally -- a single dense hyperplane removes the WHOLE concept direction and structurally cannot localize an edit to a sub-context [14][15]. The diffusion erasure cluster -- SAeUron (2501.18052, ICML 2025), SAEmnesia (2509.21379), SNCE (2509.21008) -- ablates a feature/neuron for a whole concept in TEXT-TO-IMAGE DIFFUSION, again a modality distinguisher [16][17][18].
