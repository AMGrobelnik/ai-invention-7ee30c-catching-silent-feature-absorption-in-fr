## Research Findings

See research_report.md for the full 5-work-package report and research_out.json for the machine-readable pinned facts. Executive summary with citations:

WP1 -- CHANIN 2409.14507 ABSORPTION DIAGNOSTIC (verified vs sae-spelling code). SAE config: base gemma-2-2b, Gemma Scope residual SAEs width 16k AND 65k, layers 0-17 only (metric breaks past layer 18); cross-validated on Qwen2-0.5B and Llama-3.2-1B layers 0-8 [1]. Parent latent = SAE latent whose ENCODER direction has max cosine with an LR probe (L1 decay 0.01), giving 26 first-letter latents; k=1 sparse probing gives similar results [1,4]. Feature splits via k-sparse probing KS=(1..15,20,50) at f1_jump_threshold=0.03 [1,3]. Absorption is tested on false-negative tokens (LR probe right, all k split latents miss; cap 200/letter) via integrated-gradients ablation (ig_interpolation_steps=6) on the first-letter logit-diff m=g[y]-(1/(|L|-1))*sum_{l!=y}g[l] [1,3]. _is_absorption (verbatim code logic): (1) every main-latent activation < EPS=1e-8; (2) top ablation score <= 0; (3) |top IG|-|2nd IG| >= ablation_delta_threshold=1.0; (4) top latent's DECODER-direction cosine with the probe >= probe_cos_sim_threshold=0.025 -- code: cos_sims=cosine_similarity(probe_dir, sae.W_dec, dim=-1) [1,3]. absorption_rate=num_absorptions/lr_probe_true_positives [1]. Worked/pilot example: layer 3 width 16k L0=59 SAE, 'starts with S' main latent 6510 (F1~0.81) misses _short; absorber 1085 ('short', decoder-probe cosine 0.12) fires instead; L splits into 7112 (lowercase l) + 7657 (uppercase L) [1]. SAEBench (2503.09532) re-implements as eval id 'absorption_first_letter' and reports BOTH mean_full_absorption_score (strict) and mean_absorption_fraction_score (relaxed/projection) + mean_num_split_features [6,7]. FORM-FREE version (paper Appendix A.13 = SAEBench absorption_fraction): replace IG ablation with a probe-projection test -- latent l absorbs iff tau_c < (a_hat_l . d_p)/(a . d_p), a_hat_l=(enc activation)*(W_dec[l]); works at ALL layers and replaces the undefined 'concept logit' with the probe-direction projection of the residual; needs only a parent-concept LR probe d_p trained on data DISJOINT from clustering + frozen W_dec [1]. Non-circularity: the diagnostic only SCORES already-formed specialization edges and validates the Tier-0 pilot; units are formed by content-response correlation (C-track) and anchored greedy set-cover on highest-recall latents (K-track), never by the absorption metric; the by-construction risk (holes ~ parent false-negatives) is controlled by a matched random-latent null with bootstrap CI excluding 0 [1].

WP2 -- BEYOND SPELLING? No peer-reviewed source demonstrates absorption on a NON-spelling token hierarchy: Chanin 2409.14507 empirics are first-letter only [1]; the LessWrong toy-models post says only spelling has real-model support and semantic 'X is Y' cases are illustrative [9]; SAEBench's only absorption eval is 'absorption_first_letter' [7]; Feature Hedging 2505.11756 adds a correlated-feature mode (absorption worse at WIDER SAEs, hedging worse at NARROWER) on the same spelling setup [8]; Matryoshka (2503.17547) and H-SAE (2506.01197) only mitigate absorption measured by the spelling metric [10,11]; Dense SAE Latents (2506.15679) shows numeric/year features exist in SAEs (supports a numeric parent's plausibility) but not absorption [12]. So Testbed-2 is both a generality test and a novel empirical test; do not fabricate evidence. Design: PRIMARY numeric-quantity (general 'numeric token' parent; year/percentage/date absorbers; source single-token numerics from pile-uncopyrighted; sub-context labels year/percentage/date/other) -- recommended because number-ness is dense (dense-parent/sparse-child) with objective single-token labels; ALTERNATIVE taxonomic 'is-a-country' (continent labels, but multi-token entities). Non-triviality go/no-go (all required): parent content-response recall >=0.60 (spelling parent F1~0.81 justifies the floor); >=1 (target >=2) absorber with firing Jaccard<0.10 vs parent AND sub-context precision>=0.70 AND marginal hole-coverage gain>=0.05 with bootstrap 95% CI excluding 0; else report absorption is spelling-specific, scope C3's title to spelling-type hierarchical absorption, and route generality through C1.
